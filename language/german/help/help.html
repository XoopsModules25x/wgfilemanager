Help: wgFileManager

Description

The wgFileManager module for file management in XOOPS

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgfilemanager folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Browse files and directories
  • 3 different styles
    • list
    • grouped
    • cards
  • display/hide preview and file icons
File management:
  • Upload file
  • Download file
  • Print file
  • Preview for image and pdf file (modal)
  • Edit file
  • Delete file
Directory management:
  • Create directory
  • rename directory
  • move directory by drag&drop
  • deleting directory with complete content
Permission management:
  • Global permissions
  • Permissions by directory

The wgFileManager module continues to expand, to get to the conditions to create modules, more and more sophisticated.
For this reason, I invite all developers to report and send in svn any changes or additions to this module, so that we can jointly contribute to the development
